<h1 align="center">🎭 K-Drama Flashcards</h1>

<p align="center">
  A simple, interactive React-based flashcard application designed to test your knowledge of Korean Dramas through a series of 10 trivia questions.
</p>

---

## 📌 Project Overview

This project is a lightweight web application that focuses on state management and smooth user transitions using **React Hooks** (`useState`). It was built to practice building interactive UI components with dynamic progress tracking and conditional rendering.

This project was built as part of the **[Flash Cards Project](https://roadmap.sh/projects/flash-cards)** from **roadmap.sh**.

---

## ⚙️ How It Works

The app manages a collection of 10 flashcards, allowing users to navigate through questions and toggle answers.
